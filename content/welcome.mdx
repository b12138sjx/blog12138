# 欢迎来到我的数字花园

这个博客参考了 [Obsidian](https://obsidian.md) 的核心理念：**思想是相互连接的网络，而非线性列表**。

## 特性介绍

### 双向链接

你可以在文章中用 `[[文章名]]` 的方式引用其他文章，例如：

我在 [[nextjs-notes]] 中记录了学习 Next.js 的过程，也在 [[react-basics]] 中整理了 React 的基础知识。

### 知识图谱

所有文章和它们之间的连接会被可视化为一张图谱，帮助你发现知识之间的关联。

### 反向链接

每篇文章底部都会展示**有哪些文章引用了当前文章**，让你能沿着链接双向游走。

## 开始探索

- 点击顶部导航的 **图谱** 查看所有文章的连接关系
- 使用 `Cmd/Ctrl + K` 打开全局搜索
- 点击文中的 `[[链接]]` 跳转到对应文章
